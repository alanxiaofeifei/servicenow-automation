# Design direction

A dark cinematic product reel inspired by the user's Codex and Copilot reference videos, but adapted for a GitHub public project page. The references use clean UI, negative space, floating product windows, and soft purple/blue AI cues. This composition keeps the clarity and floating UI treatment, but shifts toward a darker, more dramatic service-desk cockpit mood.

# Do

- Keep all text in English.
- Use existing repaired/local demo screenshots only.
- Make safety boundaries visible: local-first, human review, no live writes.
- Treat UI screenshots as designed objects, not raw screen recordings.
- Use music only; no narration.

# Do not

- Do not imply real ServiceNow submission is automated in this release.
- Do not show real customer/ticket/browser/session data.
- Do not use the user's bug screenshots as promotional material.
- Do not use reference-video audio or copyrighted assets.
